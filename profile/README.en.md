# purs-jp (PureScript Japan Community)

Welcome to the Japanese PureScript community, `purs-jp`!

This organization was created to support community activities in Japanese
related to the functional programming language [PureScript][purs].

[purs]: https://www.purescript.org/

## Main Activities

- Managing and sharing Japanese translations of PureScript documentation

There are currently no repositories for below, but future activities may
include:

- Creating tutorials and guides for Japanese speakers
- Maintaining PureScript packages specialized for Japanese language
  processing (morphological analysis, text normalization, etc.)

## Membership

- If you would like to join this organization, feel free to contact any
  existing member.

## Contact

- For questions or suggestions, please reach out via [Discussions][discuss]
  or through Issues in each repository.

[discuss]: https://github.com/purs-jp/.github/discussions
