# purs-jp (PureScript Japan Community)

日本のPureScriptコミュニティ`purs-jp`へようこそ！

この組織は、関数型プログラミング言語 [PureScript][purs] に関する日本語圏コミュニティ活動のために作成されました。

[purs]: https://www.purescript.org/

## 主な活動内容

- PureScriptに関するドキュメントの日本語訳の管理・共有

現時点でリポジトリ等はありませんが、今後以下に関係する活動が行われる可能性があります。

- 日本語話者向けのチュートリアルやガイドの作成
- 日本語処理に特化したPureScriptパッケージ（形態素解析、テキスト正規化など）の管理

## メンバーについて

- この組織に参加したい場合は、既存のメンバーへお気軽にご連絡ください。

## お問い合わせ

- 質問や提案がある方は、[Discussions][discuss]または各リポジトリのIssue等で連絡ください。
